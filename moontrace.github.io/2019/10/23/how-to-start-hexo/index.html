환경 설정

  • 파워쉘 관리자 권한으로 실행

  • node -v

    버전을 확인하면 패키지의 상태를 확인할 수 있다

  • npm install -g hexo-cli

    • hexo : 마크다운으로 컨버팅 해줌

  • hexo init ghblog

    • ghblog에 hexo를 넣어줘

  • npm install을 통함
